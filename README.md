# QCDGE_CI

Codes for technical validation of QCDGE_CI dataset.

## Preparing the Environment

All codes for technical validation were developed with **Python 3.11.10**.

### Package Installation

```text

pip install openbabel
pip install rdkit
pip install matplotlib
pip install ase
pip install scipy
pip install tqdm
pip install numpy
pip install pandas
pip install seaborn

```
